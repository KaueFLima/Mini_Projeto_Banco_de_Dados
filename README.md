# Mini_Projeto_Banco_de_Dados
Projeto com intuito de anonimizar informações de um banco de dados ficticio para estar de acordo com a LGPD
-----------------------------------------------------------------
Instalação
Clone ou baixe o projeto
Instale os requirements
pip install -r requirements.txt
-----------------------------------------------------------------
Como executar:
Pode ser executado de uma unica vez com o arquivo Main.py ou individualmente em cada atividade.py, fica do gosto do usuário
